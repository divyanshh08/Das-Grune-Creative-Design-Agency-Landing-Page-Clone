# Das Grüne — Creative Design Agency Landing Page Clone

A pixel-focused recreation of a creative agency landing page, built from scratch using pure HTML and CSS. This is my **first genuine project** — built to push my fundamentals in layout, typography, and visual hierarchy beyond tutorial-following.

> ⚠️ **Work in progress**: The footer section is not built yet. Coming in the next commit.

---

## 🎨 Design Credit

This project is a coded recreation of a design originally posted on Dribbble:

**[Das Grüne — Creative Design Agency Landing Page](https://dribbble.com/shots/19365923-Das-Gr-ne-Creative-Design-Agency-Landing-Page-Website)**

All visual design credit goes to the original designer. This repository contains only my own front-end code written to bring that design to life in the browser.

---

## 🛠️ Technologies Used

- **HTML5** — semantic page structure
- **CSS3** — custom layout, Flexbox, gradients, and animations
- **Google Fonts** — Poppins
- **Remix Icon** — icon library
- **Custom web fonts** — Gilroy, Helvetica Neue

---

## ✨ Features

- Bold, oversized typography-driven hero section
- Custom rounded video preview module with play button
- Infinite horizontal scrolling marquee banner
- Services section with pill-style category buttons
- Animated hover interactions on service/design rows
- Stylized multi-line quote section with accent shape blocks
- Fully responsive layout structure (in progress)

---

## 📚 What I Learned

- Translating a static design file into real, working CSS layouts
- Using Flexbox effectively for complex alignment (not just simple rows/columns)
- Working with large-scale typography without breaking layout flow
- Importing and managing custom fonts (`@font-face`) alongside Google Fonts
- Structuring CSS in a way that's easier to scale and debug
- The value of building a **complete** project end-to-end, instead of stopping halfway
